Графический дизайн 10-11 класс

В данном разделе представлены задания для учащихся, участвующих в олимпиаде по направлению “Графический дизайн”. Всего представлено три номинации: “Рисунок”, “Живопись”, “Композиция”. Выполни задание из каждой номинации и загрузи фотографии или сканы выполненных работ в форме “Отправить свои работы”

Номинация «Рисунок»

Рисунок - это изображение, выполненное
от руки, на глаз, с помощью графических
средств: линии, штриха и тонального пятна.
Чтобы приступить к выполению задания,
нажми кнопку “Открыть задание”

Номинация для школьников 10-11 классов
«Натюрморт из геометрических тел»

Конкурсное задание выполняется на листе бумаги не менее формата А3 (297х420 мм).
Материал: графитный карандаш. Общая продолжительность испытания 180 минут (3 часа).
Содержание задания: участнику олимпиады предлагается выполнить рисунок
пяти бытовых предметов на фоне трёх драпировок (без складок).

Номинация «Живопись»

Живопись - вид изобразительного искусства,
связанный с передачей зрительных образов
посредством нанесения красок на поверхность.
Чтобы приступить к выполнению задания,
нажми кнопку “Открыть задание”

Номинация для школьников 10-11 классов
«Этюд сложного натюрморта»

Конкурсное задание выполняется на листе бумаги формата А3 (297х420 мм).
Материал (по выбору): акварель, гуашь. Общая продолжительность
испытания – 240 минут (4 часа). Содержание задания: участнику олимпиады
предлагается выполнить этюд постановки натюрморта из предметов с натуры.
Количество предметов, включая муляжи: 3. Количество драпировок: три (со складками).

Номинация «Композиция»

Композиция - одна из основных категорий
художественного творчества. Сложный тип
структуры, в котором все элементы органично
связаны между собой.
Чтобы приступить к выполнению задания,
нажми кнопку “Открыть задание”

Номинация для школьников 10-11 классов
«Динамичная композиция»

Конкурсное задание выполняется на листе бумаги формата А4 (297х420 мм).Материал
(по выбору):акварель, гуашь. Общая продолжительность испытания –180 минут (3 часа).
Содержание задания: участнику олимпиады предлагается выбрать коллекцию одежды
любого дизайнера-модельера и создать модную иллюстрацию, состоящую из трех
комплектов одежды (на одном листе) из выбранной коллекции.
Работа должны быть выполнена в авторском стиле. На листе задания необходимо на-
писать имя дизанера-модельера и год создания выбранной коллекции.
