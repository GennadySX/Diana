Графический дизайн 6-7 класс

В данном разделе представлены задания для учащихся, участвующих в олимпиаде по направлению “Графический дизайн”. Всего представлено три номинации: “Рисунок”, “Живопись”, “Композиция”. Выполни задание из каждой номинации и загрузи фотографии или сканы выполненных работ в форме “Отправить свои работы”

Номинация «Рисунок»

Рисунок - это изображение, выполненное от руки, на глаз, с помощью графических средств: линии, штриха и тонального пятна. Чтобы приступить к выполению задания, нажми кнопку “Открыть задание”

Номинация для школьников 6-7 классов
«Натюрморт из геометрических тел»

Материал: графитный карандаш. Общая продолжительность испытания 240 минут (4 часа). Содержание задания: участнику олимпиады предлагается выполнить рисунок натюрморта из трёх гипсовых геометрических тел с натуры на фоне двух драпировок.

Номинация «Живопись»

Живопись - вид изобразительного искусства, связанный с передачей зрительных образов посредством нанесения красок на поверхность. Чтобы приступить к выполнению задания, нажми кнопку “Открыть задание”

Номинация для школьников 6-7 классов
«Этюд сложного натюрморта»

Конкурсное задание выполняется на листе бумаги формата А2 (597х420 мм).Материал (по выбору): акварель, гуашь. Общая продолжительность испытания –300 минут (5 часов). Содержание задания: участнику олимпиады предлагается выполнить этюд постановки натюрморта из бытовыхпредметов с натуры. Количество предметов,включая муляжи: пять. Количество драпировок: три (со складками).

Номинация «Композиция»

Композиция - одна из основных категорий художественного творчества. Сложный тип структуры, в котором все элементы органично связаны между собой. Чтобы приступить к выполнению задания, нажми кнопку “Открыть задание”

Номинация для школьников 6-7 классов
«Динамичная композиция»

Конкурсное задание выполняется на листе бумаги формата А4 (297х420 мм).Материал (по выбору) тушь, гуашь. Общая продолжительность испытания –180 минут (3 часа). Содержание задания: участнику олимпиады предлагается выполнить динамичную композицию, цвета черный и синий. Элементы: два квадрата, один прямоугольный треугольник, один равносторонний треугольник, три сектора круга, две линии, буква “Т”, буква “Е”. Композиция выполненный работы должна соответствовать поставленным перед участником задачам.
